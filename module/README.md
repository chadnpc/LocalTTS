
# [LocalTTS](https://www.powershellgallery.com/packages/LocalTTS)

🔥 A module to setup and use TTS models and few shot voice cloning

## Usage

```PowerShell
Install-Module LocalTTS
```

then

```PowerShell
Import-Module LocalTTS
# do stuff here.
```
